# 发布到阿里云镜像服务并在测试环境启动

以下命令默认发布 CPU 版镜像，端口使用 `8002`。

## 1. 本机重新构建发布镜像

```powershell
cd D:\Docker\asr-funasr-leader

docker compose build asr-leader-cpu
```

## 2. 设置镜像地址

按你现有阿里镜像仓库，建议用这个地址：

```powershell
$tag="20260528-v1"
$image="crpi-cdfock1sf5ur5v0t.cn-shanghai.personal.cr.aliyuncs.com/chenxi_project/asr/asr-leader:$tag"
```

## 3. 登录阿里云镜像服务

```powershell
docker login crpi-cdfock1sf5ur5v0t.cn-shanghai.personal.cr.aliyuncs.com
```

输入阿里云镜像服务的用户名和密码。

## 4. 打标签并上传

```powershell
docker tag funasr-leader-asr:cpu $image
docker push $image
```

## 5. 测试环境拉取镜像

在测试服务器上执行：

```bash
docker login crpi-cdfock1sf5ur5v0t.cn-shanghai.personal.cr.aliyuncs.com
docker pull crpi-cdfock1sf5ur5v0t.cn-shanghai.personal.cr.aliyuncs.com/chenxi_project/asr/asr-leader:20260528-v1
```

## 6. 测试环境启动

在测试服务器创建目录：

```bash
mkdir -p /data/asr-leader
cd /data/asr-leader
mkdir -p data
```

创建 `docker-compose.yml`：

```yaml
services:
  asr-leader-cpu:
    image: crpi-cdfock1sf5ur5v0t.cn-shanghai.personal.cr.aliyuncs.com/chenxi_project/asr/asr-leader:20260528-v1
    ports:
      - "8002:8000"
    environment:
      - ASR_DEVICE=cpu
      - MODELS_DIR=/app/models
      - DATA_DIR=/data
      - LEADER_THRESHOLD=0.62
      - LEADER_SPEAKER_THRESHOLD=0.35
      - LEADER_SEGMENT_THRESHOLD=0.50
      - LEADER_SCORE_MARGIN=0.03
      - LEADER_MIN_SEGMENT_SECONDS=1.0
    volumes:
      - ./data:/data
    restart: unless-stopped
```

启动：

```bash
docker compose up -d
docker compose logs -f
```

## 7. 验证

```bash
curl http://localhost:8002/health
```

正常返回：

```json
{
  "status": "ok",
  "model_loaded": true,
  "device": "cpu",
  "leader_count": 0,
  "models_dir": "/app/models"
}
```

接口文档：

```text
http://测试服务器IP:8002/docs
```

## 8. 客户测试流程

先上传领导声纹：

```bash
curl -X POST http://测试服务器IP:8002/leaders/enroll \
  -F "file=@leader.wav" \
  -F "leader_id=leader_001" \
  -F "name=领导姓名"
```

再上传会议音频识别：

```bash
curl -X POST http://测试服务器IP:8002/transcribe \
  -F "file=@meeting.wav" \
  -F "num_speakers=10" \
  -F "identify_leaders=true"
```

领导声纹数据会保存在测试服务器：

```text
/data/asr-leader/data/leaders.json
```
